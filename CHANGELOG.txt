Alfresco 6.x-1.1-dev, 2010-01-xx
============================

Nuevas funcionalidades:

- Subida de documentos al repositorio de Alfresco desde Drupal
- Sistema de autenticación mediante credenciales de usuarios de Drupal
- Búsqueda en el repositorio de Alfresco desde Drupal
- Estadísticas de descargas de elementos de Alfresco
- Actualizada la biblioteca de Alfresco PHP API (Alfredo-PHPAPI-R2)
- Menú Alfresco:
  * Formulario de subida de contenidos al repositorio Alfresco
  * Listado de elementos de Alfresco creados en Drupal
  * Navegador por el repositorio de Alfresco (Alfresco Browser)
  * Importación de contenidos desde un espacio (Alfresco Importer)
- Módulo Alfresco Browser:
  * Eliminada la biblioteca Ext JS que se distribuía con el módulo, ahora es
    una depedencia externa que debe ser instalada independientemente
  * Operaciones de subida y borrado de ficheros
  * Path al espacio de inicio configurable
  * Número de elementos por página configurable
  * Caché (activación/desactivación) configurable
  * Añadidos botones para subir y borrar desde el navegador
  * Añadida máscara de carga
  * Añadida internacionalización
- Integraciones con módulos de terceros:  
  * Integración con Admin Menu
  * Integración con Admin Theme
  * Integración con Vertical Tabs

Mejoras y correcciones:

- Solución para el fallo del NameSpace del dictionary-service.wsdl
- Mejorado el registro de errores de las conexiones SOAP
- Mejorada la detección de tipos de contenidos de espacios de Alfresco
- Mejorada la integración con Views
- Actualizada las tradaducciones
- Arreglados muchos pequeños fallos

Alfresco 6.x-1.1-dev, 2010-01-xx
============================

Nuevas funcionalidades y mejoras:

- Implementada la subida de documentos al repositorio de Alfresco desde Drupal

- Nuevo menú Alfresco, desde donde se puede acceder:
  * Formulario de adición de contenido al repositorio de Alfresco
  * Navegador por el repositorio de Alfresco
  * Listado de elementos de Alfresco de Drupal

- Mejoras del módulo Alfresco Browser:
  * Añadido máscara de carga
  * Personalización del Espacio de inicio
  * Mejorada la detección de tipos de contenidos Folders
  * Implementada la subida de documentos
  
- Mejorada la integración con Views
  * Añadida una vista por defecto

- Añadida integración con Admin Theme

Cambios para desarrolladores:
- Actualizada la biblioteca de Alfresco PHP con Alfredo Common (enlace)
- Eliminada la biblioteca Ext JS que se distribuía con el módulo, ahora es una depedencia.

Otros cambios:
- Actualizada la traducción
- Añadida una solución para el fallo del NameSpace del dictionary-service.wsdl
- Fixed a variety of other bugs.

Alfresco 6.x-1.0, 2009-03-23
============================
- alfresco_browser: New module which allows users to visualize, search, browse
  and retrieve nodes of the Alfresco repository.
- alfresco_attach: Fixed permission checking when remove attach items.
- alfresco_attach: Simplified attach existing nodes.
- Removed unnecessary files from Alfresco PHP Library.
- General code clean-up.
- Minor fixes.
